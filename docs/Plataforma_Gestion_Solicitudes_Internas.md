## DOC00 - Descripción General
### Proyecto: Plataforma de Gestión de Solicitudes Internas

El proyecto tiene como objetivo centralizar los procesos operativos internos en una única aplicación web, eliminando la dependencia de correos electrónicos, hojas de cálculo y comunicaciones informales.

#### Funcionalidades principales:
- Creación y consulta de solicitudes.
- Adjuntar documentación.
- Notificaciones y seguimiento de procesos.
- Gestión por roles y permisos.

#### Arquitectura técnica:
- **Frontend:** React.
- **Backend:** APIs REST.
- **Autenticación:** Corporativa con gestión de roles.

#### Flujo de trabajo:
1. Creación de solicitud.
2. Revisión inicial.
3. Resolución por el equipo responsable (aprobación, rechazo, reasignación).
4. Registro de cambios para trazabilidad.

#### Metodología de desarrollo:
- Uso de Jira para gestión de tareas.
- Desarrollo en ramas independientes.
- Revisión mediante pull requests.
- Despliegue en entornos de validación.
